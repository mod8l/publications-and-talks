# My Public Work

This section is reserved for my own papers, patents, and talks. The templates below use YAML frontmatter so they can be picked up by `make index`.

When I add a real entry, I will remove the placeholder row from the main README or the section index.

## Templates

- [Paper template](./papers/template.md)
- [Patent template](./patents/template.md)
- [Talk template](./talks/template.md)

## Adding a new entry

1. Copy the relevant template into a new file or directory.
2. Fill in the YAML frontmatter and body.
3. Run `make index` from the repository root to regenerate the README.
