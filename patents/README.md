# Patents

A curated list of patents and patent applications.

| Filing Date | Title | Inventors | Number | Links |
|-------------|-------|-----------|--------|-------|
| YYYY-MM-DD | [Title of your patent](./template.md) | [Your Name], [Co-inventor Name] | [Patent number] | [Details](./template.md) |

## Template

Use the [patent template](./template.md) when adding a new patent entry.

## Add a new patent

1. Copy [template.md](./template.md) to a new file in `patents/`.
2. Replace all placeholder values with real content.
3. Run `make index` to regenerate the main README.
