# Streaming Feature Stores for Real-Time Recommendation Systems

**gadsosa, A. R. Chen, M. J. Okafor**  
*RecSys 2021*

[DOI](https://dl.acm.org/doi/10.1145/3460231.3474233)

## Abstract

We present a streaming feature store architecture designed for sub-100ms feature freshness in large-scale recommendation systems. Our design decouples feature computation from feature serving, uses deterministic event-time joins, and exposes a unified API across batch and stream contexts. We evaluate the system on a production e-commerce dataset and show that online A/B metrics improve by 4.2% when feature lag is reduced from minutes to seconds, while serving cost increases by less than 12%.
