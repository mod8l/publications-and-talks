# Papers

A curated list of papers, preprints, and technical publications.

| Year | Title | Authors | Venue | Links |
|------|-------|---------|-------|-------|
| YYYY | [Title of your paper](./template.md) | [Your Name], [Co-author Name] | [Venue] | [Paper](./template.md), [DOI](./template.md) |

## Template

Use the [paper template](./template.md) when adding a new publication.

## Add a new paper

1. Copy [template.md](./template.md) to a new file in `papers/`.
2. Replace all placeholder values with real content.
3. Run `make index` to regenerate the main README.
