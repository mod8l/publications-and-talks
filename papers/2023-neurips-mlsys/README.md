# Adversarial Robustness of Production Computer Vision Pipelines

**gadsosa, L. Petrov, S. Nakamura**  
*NeurIPS 2023 Workshop on Machine Learning Systems*

[arXiv](https://arxiv.org/abs/2312.04567)

## Abstract

We study the robustness of production computer vision pipelines against patch-based adversarial attacks under realistic constraints: fixed inference latency, black-box model access, and post-processing steps such as resizing and compression. We introduce a lightweight augmentation strategy that reduces attack success rate by 34% on average across three public benchmarks without retraining the underlying model. Our code and evaluation harness are available as an open-source benchmark.
